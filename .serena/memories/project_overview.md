# billing_go 项目概览

## 项目目的
billing_go 是一个用 Go 语言编写的 billing 验证服务器，主要用于游戏账号认证和计费管理。

## 主要功能
- 游戏账号登录验证
- 点数（积分）查询和管理
- 账号注册（支持自动注册）
- 在线玩家管理
- 活动奖励处理
- 支持经典版和怀旧版两种 billing 类型

## 运行环境
- Linux (内核版本 2.6.32+)
- Windows 10, Server 2016+
- 可作为前台进程、后台守护进程或 systemd 服务运行

## 配置支持
- 支持 YAML 和 JSON 配置文件格式
- 配置项包括服务器IP、端口、数据库连接、自动注册、IP限制等
